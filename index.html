Bibleseeds

Online Social Bible Commentary & Study Tools

Download this project as a .zip file Download this project as a tar.gz file

Welcome to BibleSeeds.

This is the initial implementation for online social bible commentary & study tools. The idea is to be able to share your comments / thoughts / ideas / notes / sermons - and can be cross referenced with Bible verses.
